# Meta 광고 플레이북

> mintorain 검증 패턴. funnel-marketing 스킬이 참조합니다.

## 광고 계정 구조

```
캠페인 (Campaign)
├── 목표: 전환 (리드 수집)
├── 예산: 일 5만원~10만원 (테스트)
│
├── 광고 세트 A: 관심사 타겟
│   ├── 30-45세, 여성+남성
│   ├── 관심사: 1인창업, 부업, 온라인비즈니스
│   └── 광고 소재 3개 (A/B/C)
│
├── 광고 세트 B: 유사타겟 1%
│   ├── 구매 고객 기반
│   └── 광고 소재 3개 (A/B/C)
│
└── 광고 세트 C: 리타겟팅
    ├── 웹사이트 방문자 30일
    ├── 영상 50% 이상 시청자
    └── 광고 소재 3개 (D/E/F)
```

## 15초 영상 광고 스크립트 구조

```
[0-3초] 훅: 스크롤 멈추게 하는 한 마디
  예: "코딩 1도 모르는 제가 3일 만에 웹사이트를 만들었습니다"
  예: "개발자 없이 매출 3배 올린 비밀?"

[3-8초] 문제 공감:
  예: "홈페이지 외주 맡기면 300만원, 수정할 때마다 돈..."

[8-12초] 해결책:
  예: "바이브코딩으로 AI한테 말만 하면 웹사이트가 뚝딱"

[12-15초] CTA:
  예: "지금 무료 체험 신청하세요. 링크 클릭!"
```

## 리드 폼 최적화

### 필드 구성 (최소화)
```
필수: 이름, 전화번호
선택: 이메일 (미기입 시 카카오로 대체)

⚠️ 필드가 많을수록 전환율 하락
⚠️ 전화번호 > 이메일 (한국 시장 특성)
```

### CTA 문구 패턴 (검증된 순서)
```
1순위: "무료 체험 신청하기" (가장 높은 전환율)
2순위: "무료 PDF 받기"
3순위: "지금 시작하기"
피하기: "제출", "구매하기" (저항감)
```

## 성과 분석 기준

| 지표 | 좋음 | 보통 | 개선 필요 |
|------|------|------|-----------|
| CTR | 2%+ | 1-2% | 1% 미만 |
| CPL (리드 단가) | 3,000원 이하 | 3,000-5,000원 | 5,000원 초과 |
| LP 전환율 | 15%+ | 8-15% | 8% 미만 |
| ROAS | 300%+ | 200-300% | 200% 미만 |

## 트러블슈팅

### 광고 성과 하락 시 체크리스트
```
□ 타겟 피로도 확인 (빈도 3 이상이면 타겟 교체)
□ 소재 피로도 확인 (CTR 하락 추이)
□ 랜딩페이지 로딩 속도 확인 (3초 이내)
□ 리드 폼 이탈률 확인
□ 경쟁사 새 광고 확인 (Ad Library)
□ 시즌/시기 요인 검토
```
