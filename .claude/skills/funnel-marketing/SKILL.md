# 퍼널 마케팅 설계 스킬

## 트리거 조건
- "퍼널", "리드", "전환", "마케팅" 키워드 감지 시
- 광고, ROAS, CPA 관련 요청 시

## 퍼널 설계 프레임워크 (mintorain 20년 노하우)

### 4단계 퍼널 구조
```
┌─────────────────────────────────────────┐
│  1단계: ATTRACT (유입)                   │
│  Meta 광고 / 유튜브 / 블로그 / 쓰레드     │
│  목표: 타겟 고객의 관심 끌기              │
│  KPI: CPC, CTR, 도달률                  │
├─────────────────────────────────────────┤
│  2단계: CAPTURE (획득)                   │
│  랜딩페이지 → 리드 수집 폼               │
│  무료 혜택: PDF, 미니강의, 체험판          │
│  KPI: 전환율, CPL (리드 단가)            │
├─────────────────────────────────────────┤
│  3단계: NURTURE (너처링)                 │
│  이메일 시퀀스 + 카카오 알림톡            │
│  가치 제공 → 신뢰 구축 → 긴급성 부여      │
│  KPI: 오픈율, 클릭률, 응답률             │
├─────────────────────────────────────────┤
│  4단계: CONVERT (전환)                   │
│  세일즈페이지 → 결제                     │
│  한정 혜택, 보증, 사회적 증거             │
│  KPI: 전환율, ROAS, 고객생애가치          │
└─────────────────────────────────────────┘
```

### Meta 광고 전략 (검증된 패턴)

#### 15초 영상 광고 프레임워크
```
0-3초: 훅 (스크롤 멈추게 하는 질문/충격)
3-8초: 문제 공감 + 해결 힌트
8-12초: 솔루션 소개 (바이브코딩)
12-15초: CTA (지금 무료로 시작하세요)
```

#### 타겟팅 설정
```json
{
  "audience": {
    "age": "30-55",
    "interests": ["1인창업", "온라인비즈니스", "부업", "마케팅"],
    "exclude": ["현직 개발자", "IT 종사자"],
    "lookalike": "구매 고객 1% 유사타겟"
  }
}
```

### 이메일 시퀀스 설계
```
Day 0: 환영 + 무료 혜택 전달
Day 1: 스토리 (왜 바이브코딩을 만들었나)
Day 2: 가치 제공 (바이브코딩 팁 1)
Day 3: 사회적 증거 (수강생 후기)
Day 5: 가치 제공 (바이브코딩 팁 2)
Day 7: 소프트 셀 (강의 소개)
Day 10: 긴급성 (마감 임박)
Day 14: 라스트 콜 + 특별 혜택
```

### n8n 자동화 연동 포인트
```
[Meta Lead Ad] → [n8n Webhook]
    → [Google Sheets 저장]
    → [Solapi 알림톡 발송]
    → [ConvertKit 이메일 시퀀스 시작]
    → [Slack 알림 (새 리드)]
```

## 전환율 벤치마크
| 단계 | 목표 | 최소 |
|------|------|------|
| 광고 CTR | 2%+ | 1% |
| LP 전환율 | 15%+ | 8% |
| 이메일 오픈율 | 30%+ | 20% |
| 최종 구매 전환 | 5%+ | 2% |
